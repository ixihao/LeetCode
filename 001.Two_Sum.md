### 1.1 Basic method
**Idea**:  
**Time Complexity**:  
``` python
class Solution:
    def twoSum(self, nums, target):
        # Basic Method
        for i in range(len(nums)):
            for j in range(i+1, len(nums)):
                if nums[i] + nums[j] == target:
                    return[i,j]
```
